상호작용 물체의 미래 예측

연구 배경

개체 간 관계가 시시각각 변하는 동적 상호작용 시스템에서 개체 간 관계를 GNN을 이용하여 비감독 방식으로 발견하고, 시스템의 미래 상태를 예측
기존 모델 : DSLR

DSLR은 GNN을 활용하여 상호작용 시스템의 미래를 예측하는 모델. DSLR은 정적 관계 시스템에 대한 예측 성능이 우수함. 하지만 동적 관계 시스템에 대한 미래를 잘 예측하지 못함.

해결 시도 1: Replay Memory

데이터를 학습시키는 과정에서, 본래 데이터를 그대로 학습 시키지 않고, 과거의 학습 데이터중 가장 유사한 데이터를 가져와서 학습시키는 기법
해결 시도 2: Data Augmentation

Encoder로 들어가는 위치값을 랜덤한 만큼 바꿔줌 Decoder로 들어가는 위치값은 그대로 이용

음성데이터 합성 모델 연구

연구 주제

Zero Speech Challenge : 사람이 말하는 내용을 파악한 뒤, 다른 사람이 말하는 것처럼 음성 발화 데이터를 합성하는 기술
VQ-VAE 모델과 배경

Autoencoder의 일종으로 encoder와 decoder의 두 부분으로 이루어짐, 학습된 표현들은 음성 콘텐츠만 포함하도록 조정되기 때문에 WaveNet decoder를 사용해서 인코더에서 손실된 정보들을 추론하도록 구성
해결법 연구: GE2E

GE2E는 이전의 화자 검증 모델과 달리 새로운 손실함수를 사용해 스피커 임베딩 벡터를 학습함. 화자가 같은 기존 스피커 벡터들의 무게중심과 가깝도록, 화자가 다른 가장 가까운 스피커 벡터의 무게중심으로부터 멀어지도록 training하므로 화자 검증 능력이 기존의 모델들보다 뛰어남.
